# Kraken CLI - Official Command-Line Tool

## What is Kraken CLI?

Kraken CLI is Kraken's official command-line interface for trading. It's a standalone tool that:
- Runs locally on your machine
- Connects to Kraken's API
- Provides terminal-based trading interface
- Works even if the website blocks you

## Key Features

- **Terminal-based trading** - No browser needed
- **Direct API access** - Bypasses website restrictions
- **Scriptable** - Can be automated
- **Open source** - Can inspect the code

## Why This Matters for Your Bot

**Good news:** If Kraken CLI works, it means:
1. The API itself is accessible (not geo-blocked)
2. Only the website is blocked
3. Your trading bot can use the same API

**This changes everything.** The website block doesn't matter if the API works.
